# booklog-ch3

project booklog-ch3

- Spring Boot 2
- Bootstrap 4
  - https://getbootstrap.com/docs/4.1/examples/blog/

## Quick Start
Pre Installed
- JDK 1.8 (or Java 10)
- Maven 3.5
- Git

Run
```
git clone https://github.com/origoni/booklog-ch3.git
cd booklog-ch3
mvn spring-boot:run
```
